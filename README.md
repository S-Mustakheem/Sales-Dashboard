# Sales-Dashboard
Power BI Sales Dashboard Project
A professional Sales Dashboard developed using Power BI to analyze sales performance, customer insights, product trends, and regional business growth. This project demonstrates end-to-end Power BI development including data cleaning, data modeling, DAX calculations, and interactive dashboard creation.

Description
This dashboard was built using Power Query for data cleaning and transformation, Data Modeling with Relationships, One-to-Many Relationships, Fact & Dimension Tables, Star Schema Design, DAX Measures, and interactive visualizations. The project simulates a real-world business sales reporting system used in companies for decision-making and performance tracking.

Features
Total Sales KPI|Total Orders KPI|Total Quantity KPI|Product-wise Sales Analysis|Region-wise Sales|Distribution|Customer Sales Details|Interactive Slicers & Filters|Professional Dashboard Layout.

Dataset Information
The project contains the following tables:
Sales → Fact Table
